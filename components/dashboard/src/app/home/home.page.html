Bobbycar is a Red Hat solution pattern for simulating connected vehicles traveling through configuration zones.
This architecture can serve as a foundation for automotive hybrid cloud and OTA deployments.
Each time a simulated vehicle enters a new zone, it receives configuration and service information via an OTA update.

 High Level Architecture

This solution pattern aims to showcase an end-to-end workflow, from connecting vehicles and in-vehicle platform components to the cloud backend, processing telemetry data in batch or as stream, training AI/ML models and deploying containers through a DevSecOps pipeline and by leveraging GitOps to the edge.
